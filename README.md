# 🚀 Netflix_azure_databricks_project
Building a modern data platform on Azure using Medallion Architecture, incremental processing, and scalable analytics.

Welcome to azure_data_engineering_project. This project demonstrates the design and implementation of a modern data platform using Azure services. It covers end-to-end data ingestion, incremental processing with streaming, data transformation, and analytics. The platform is optimized for scalability, governance, and business intelligence use cases.

## 🔍 Specifications

### Data Sources: 
Simulated REST API ingestion using GitHub-hosted CSV files
Data Ingestion: Dynamically load multiple files into ADLS Gen2 using parameterized pipelines in Azure Data Factory
Incremental Processing: Implement streaming-based ingestion using Databricks Autoloader
Data Quality: Cleanse and standardize data (handle nulls, duplicates, schema enforcement)
Data Architecture: Implement Medallion Architecture (Bronze, Silver, Gold layers)
Data Governance: Use Unity Catalog for access control, metadata, and secure storage
Storage: Use Delta Lake format for reliability and performance
